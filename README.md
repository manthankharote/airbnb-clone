Title: Wanderlust - Full Stack Hotel Booking App

Short Description: A robust full-stack web application inspired by Airbnb, allowing users to list, browse, and review vacation rentals. Built using the MVC (Model-View-Controller) architecture for scalability and maintainability.

Key Features:

Authentication & Authorization: Secure login/signup using Passport.js with session-based authentication.

CRUD Operations: Users can create, read, update, and delete property listings.

Image Storage: Integrated Cloudinary with Multer for seamless image uploads.

Interactive Maps: Implemented Mapbox SDK for geocoding and displaying listing locations.

Review System: Users can leave ratings and reviews, with backend validation ensuring only the author can delete them.

Error Handling: Custom middleware for validation (Joi) and error handling to ensure application stability.

Tech Stack:

Frontend: EJS, Bootstrap, HTML5, CSS3

Backend: Node.js, Express.js

Database: MongoDB (Atlas), Mongoose

Tools: Mapbox API, Cloudinary, Joi, Helmet
